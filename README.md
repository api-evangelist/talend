# TalendThis is a repo for managing machine readable artifacts for the [Talend APIs](http://www.talend.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Talend APIs](http://www.talend.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.talend.com)- [Developer](http://www.talend.com)- [Twitter](https://twitter.com/talend)- [GitHub](https://github.com/Talend)The [APIs.json](https://github.com/api-evangelist/talend/blob/master/apis.json) index provides links to all the relevant links for the Talend APIs, including the OpenAPI and Postman if they are present.
